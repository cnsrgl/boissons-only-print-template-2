Custom template for printing beverage-focused receipts with BizPrint for WooCommerce.

== Description ==

The Boissons Only Print Template is designed specifically for beverage businesses using WooCommerce and BizPrint. This template provides a clean, professional layout optimized for beverage orders.

= Features =
* Clean, professional design
* Optimized layout for beverage orders
* Compatible with the latest WooCommerce and BizPrint versions
* Customizable through BizPrint settings
* Support for both HTML and Plain text formats
* Dynamic category filtering - choose which product category to display
* Optional customer information display
* Custom header text option
* Multi-language support
* Developer-friendly hooks for extending functionality

== Installation ==

1. Upload the `boissons-only-print-template` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to BizPrint > Settings > Location > Select your printer location
4. Choose "Boissons Only" from the template dropdown
5. Save your settings

== Frequently Asked Questions ==

= Does this template work with the latest WooCommerce? =
Yes, this template is compatible with the latest version of WooCommerce.

= Can I customize the template? =
Yes, you can customize the template through the BizPrint settings:
- Choose which product category to filter (default: boissons)
- Show/hide customer information
- Set custom header text
- For more advanced customization, you can modify the template files directly.

= Can I use this for categories other than beverages? =
Yes! In the BizPrint location settings, you can change the category slug to any product category you want to filter.

= What if no products are showing? =
1. Enable Debug Mode in the template settings
2. Check the available category slugs list in the settings
3. Make sure you're using the exact category slug (not the category name)
4. Check the HTML comments in the print preview for debug information
